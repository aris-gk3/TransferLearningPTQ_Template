The 5 datasets used can be found here:

1. [CatsDogs](https://www.kaggle.com/datasets/arisiliasgoutis/catsdogsconv)
2. [Oxford Flowers](https://www.kaggle.com/datasets/arisiliasgoutis/oxford-flowers-17-restructured)
3. [UCMerced Land Use](https://www.kaggle.com/datasets/arisiliasgoutis/uc-merced-21-restructured-and-converted)
4. [Oxford Pets](https://www.kaggle.com/datasets/arisiliasgoutis/oxdord-pets-restructured)
5. [Stanford Dogs](https://www.kaggle.com/datasets/arisiliasgoutis/stanford-dogs-restructured)

Note: Minor preprocessing has been applied to the public datasets before uploading to Kaggle.
